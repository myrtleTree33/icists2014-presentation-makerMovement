## Installation

### Development

You will need NodeJS (`bower`, `grunt`), Ruby, and SaSS installed.  Then install dependencies by running:

	$ bower install
	$ grunt server

### Distribution

Files must be located on a server.  To generate files, run:

	$ grunt dist
